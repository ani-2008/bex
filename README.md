# BEX #

## Overview ##

BEX is a Boolean Expression evaluator, Learnt about parsing and lexing while building this :D

## Usage ##

- Evaluates Boolean expression with boolean values (1 and 0)
- `+` for OR
- `.` for AND
- `~` for NOT
